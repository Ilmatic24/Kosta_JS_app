## Thema
Variablen in JavaScript

## Ziele
- `var`, `let`, `const` unterscheiden und korrekt anwenden

## Notizen
### Variablendeklaration
- `var`: funktionsweit, hoisted, erneute Deklaration erlaubt
- `let`: blockweit, hoisted (TDZ), erneute Deklaration nicht erlaubt
- `const`: blockweit, hoisted (TDZ), erneute Deklaration nicht erlaubt, Bindung bleibt konstant

### Datentypen
- Primitive: `string`, `number`, `boolean`, `null`, `undefined`, `bigint`, `symbol`
- Referenztypen: `object` (Arrays, Funktionen, Objekte)
- `string`: Text, z. B. `"Hallo"`
- `null`: bewusst gesetzter “leer”-Wert
- `undefined`: Wert wurde (noch) nicht zugewiesen

### Template Strings
Template Strings (Backticks) erlauben Interpolation und Multi-Line.

Beispiel: Interpolation
```js
const name = "Kosta";
const msg = `Hallo ${name}!`;
```
Beispiel: Ausdruck
```js
const sum = `2 + 3 = ${2 + 3}`;
```
Beispiel: Multi-Line
```js
const text = `Zeile 1
Zeile 2`;
```
